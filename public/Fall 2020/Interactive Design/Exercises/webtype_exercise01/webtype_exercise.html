Type size note (from w3schools.com): 1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px. In the heading above, the text size in em is 1.5 em, or 150% of the default size.

Practice indenting text using the text-indent property (usually specified in pixels or ems). Like this:

This paragraph has been assigned a class selector of .indent. The .indent class is styled like this: .indent { text-indent: 20px; } Notice that only the first line of this paragraph is indented. To indent all lines in a block of text, use a block quote or an unordered list.

Practice changing the text-align property for paragraphs and headings like this:

  • h1 {
  • text-align: center; }
  • p {
  • text-align: justify; }

Note: text-align: left is easiest to read. Justified type can create awkward gaps within a line, and rivers of space in blocks of type.

Practice changing the font-family list stacks. Remember that when choosing a typeface, a browser can only display fonts that are installed on that computer. Typefaces that are likely to be installed on most computers include serif fonts: Georgia, Times, and Times New Roman; sans-serif fonts: Arial, Verdana, and Helvetica; and monospace fonts: Courier and Courier New.

List font-families in your CSS like this:

  • Body {
  • font-family: Arial, Verdana, sans-serif;
  • font-size: 12px;}
  • h1 {
  • font-size: 200%;}
  • h2 {
  • font-size: 1.3em;}

In the example above, a browser will display body copy in Arial unless Arial is not installed. If Arial is not installed on that computer, Verdana will be used; if Verdana is not installed, then any sans-serif available will be displayed.
